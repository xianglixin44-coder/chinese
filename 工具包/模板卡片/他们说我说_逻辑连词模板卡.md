# 他们说/我说 · 逻辑连词模板卡

> 来源：《They Say / I Say》Gerald Graff
> 用法：每次写议论文段落时，从以下三类中各选至少 1 个模板嵌入过渡处。

---

## 🅰 引入对立观点（"他们说"）

**用法：** 开篇或转折时，先承认对方观点，展示你有全局视野（Ethos 人格诉求）。

| 编号 | 模板句式 | 使用场景 |
|---|---|---|
| A1 | **"诚然，[对立观点]在某种程度上具有其合理性，例如……；然而，如果我们审视其背后的深层逻辑，就会发现……"** | 标准的"先立后破"结构 |
| A2 | **"有人或许会主张：[对立观点]。这种看法并非全无道理，但它忽略了一个关键前提——……"** | 委婉反驳，适合考试稳妥风格 |
| A3 | **"站在[持方]的立场来看，[对立观点]确实有其依据。但当我们把视野放宽到[更大的层面]，就会发现……"** | 从小争论上升到宏大主题 |
| A4 | **"不可否认，[对立观点]在[某个局限范围]内是成立的。然而，这恰恰暴露了它的盲区——……"** | 先限定再突破，逻辑层次感强 |

### ✍ 示例（高考话题：快与慢）

> **诚然，**「快节奏生活」在效率至上的现代社会有其合理性，毕竟效率就是生命；**然而，**如果我们审视人类精神生活的深层需求，就会发现——一味求快正在吞噬我们思考与沉淀的空间。

---

## 🅱 推进己方论证（"我说"）

**用法：** 从"破"转入"立"，展现你的逻辑推理（Logos 理性诉求）。

| 编号 | 模板句式 | 逻辑功能 |
|---|---|---|
| B1 | **"这并非偶然的巧合，而是……的必然结果。究其本质，是因为……"** | 因果分析，推本质 |
| B2 | **"事实上，[论点]不仅体现在[A层面]，更深刻地反映在[B层面]。二者的内在关联在于……"** | 层次递进，纵深推进 |
| B3 | **"如果我们将[现象 A]与[现象 B]进行类比，就会发现它们在逻辑结构上共享一个核心特征——……"** | 类比论证，化抽象为具体 |
| B4 | **"从更根本的层面来看，[论点]之所以成立，是因为它触及了一个被普遍忽略的前提——……"** | 揭示前提，升维打击 |

### ✍ 示例（接上例）

> **这并非偶然的巧合，**而是现代社会资本逻辑渗透到生活方方面面的必然结果。**究其本质，**是因为"效率至上"的评价体系已经从工厂车间蔓延到了教育、医疗甚至家庭生活。

---

## 🅲 价值升华与感性收束（"所以呢"）

**用法：** 结尾或段尾，将理性分析升华为价值关怀（Pathos 情感诉求）。

| 编号 | 模板句式 | 情感功能 |
|---|---|---|
| C1 | **"由此，我们不仅看到了……，更应当警惕其背后……的深层危机。"** | 警示语气，适合收尾 |
| C2 | **"或许，最好的答案并不在于[二选一]，而在于我们在[更高维度]上寻得一种平衡——……"** | 辩证收束，体现成熟思维 |
| C3 | **"当我们重新审视[讨论的话题]时，会发现它最终指向了一个更本质的问题：我们究竟想要怎样的……？"** | 开放式结尾，引人深思 |
| C4 | **"归根结底，[论点]所关切的，不仅仅是……的问题，更关乎每一个人在[时代大背景之下的安身立命]。"** | 宏大升华，适合结尾段 |

### ✍ 示例（接上例）

> **由此，我们不仅看到了**效率崇拜对个人时间的侵占，**更应当警惕其背后**将"人"降格为"生产要素"的深层危机。**归根结底，**"快与慢"的争论所关切的，不仅是时间管理的问题，更关乎每一个人在加速时代里如何安身立命。

---

## 🃏 段落组装指南

一个完整的议论文段落 ≈ **A（引入对立） + B（推进己方） + C（升华收束）**

```
┌─────────────────────────────────────────────────────────┐
│   🅰 诚然，[对方观点]……然而，[转向己方]……            │
│                                                         │
│   🅱 这并非偶然……究其本质……                           │
│                                                         │
│   🅲 由此，我们不仅看到了……更应当警惕……               │
└─────────────────────────────────────────────────────────┘
```

### 初学者练习（第 1–2 周）

1. 每天选一个话题（如：手机该不该进校园）
2. 只用 **A1 + B1 + C1** 写一个 150 字段落
3. 目标是"套得自然"，而不是"套得生硬"

### 进阶练习（第 3–4 周）

1. 同一个段落，尝试不同的 A/B/C 组合
2. 删掉模板标记词，只保留逻辑结构
3. 对照优秀范文，用模板拆解别人的段落结构

---

*模板是拐杖，熟练后扔掉拐杖走路，才是真正的内化。*
