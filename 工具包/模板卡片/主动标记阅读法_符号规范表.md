# 主动标记阅读法 · 符号规范表

> 来源：《如何阅读一本书》分析阅读法 改编
> 用法：将此表打印/贴在书桌前。每次阅读现代文时，**手不离笔**，看到对应内容就标符号。

---

## 📋 标准符号系统

| 符号 | 名称 | 在什么情况下用 | 批注示例 |
|---|---|---|---|
| **[ ]** | 核心概念框 | 出现关键词、核心意象、被反复提及的词 | `[工匠精神]` |
| **/** | 层次分割线 | 段落内事实→分析→结论的逻辑转换处 | `事实描述/ 作者分析/ 结论` |
| **?** | 困惑标记 | 作者写得反常、矛盾、或你读不懂的地方（往往就是考点） | `? 为什么作者在这里写了一只猫?` |
| **★** | 主旨句标记 | 每一段的结论句、全文的中心论点句 | `★ 由此可见，传统不是包袱而是资源` |
| **~~~** | 论据波浪线 | 作者用来支撑观点的例子、数据、引用 | `~~~ 据统计，2019年……` |
| **→** | 逻辑箭头 | 因果关系、转折关系、递进关系的标志词 | ` → 因此 → 然而 → 更进一步` |
| **▲** | 词/句品味 | 用得精妙或值得模仿的词句 | `▲「沉默在喧嚣中反而震耳欲聋」` |
| **×** | 逻辑漏洞 | 偷换概念、以偏概全、因果倒置 | `× 这个例子不足以证明论点` |

---

## 📐 批注三层法

每次标完符号后，在文末留白处写 **3 句批注**：

```
1️⃣ 概括层：这篇文章在说什么？（1 句话）
   例：本文讨论了互联网时代阅读碎片化的利弊。

2️⃣ 结构层：作者是怎么说的？（逻辑框架）
   例：先提出现象→再分析原因→最后呼吁回归深度阅读。

3️⃣ 批判层：我同意/不同意什么？为什么？
   例：我同意碎片化的弊端，但作者忽略了碎片化阅读也可能增加信息广度。
```

## 🖨️ 使用流程

```
┌──────────────────────────────────────────────────┐
│  ① 第一遍快速通读（5 分钟）                       │
│    → 用 / 分割段落层次                             │
│    → 用 [ ] 圈出反复出现的核心词                   │
│                                                    │
│  ② 第二遍细读（15 分钟）                           │
│    → 用 ★ 标出主旨句                               │
│    → 用 ~~~ 画出关键论据                           │
│    → 用 ? 标记费解之处                             │
│                                                    │
│  ③ 第三遍整合（5 分钟）                           │
│    → 用 → 画出逻辑链条                             │
│    → 在文末写 3 句批注                             │
│    → 口头复述一遍全文主旨                         │
└──────────────────────────────────────────────────┘
```

## 💡 进阶技巧

### 同一篇文章标记两轮
第一轮用铅笔轻标，第二轮用红笔修正和补充。两次标记之间的差异，就是理解的提升。

### 对比标记
把两篇观点相反的文章放在一起读，用 **★(A)** 和 **★(B)** 分别标记各自论点，然后用 **×** 标出冲突点，这是训练批判性思维的高效方式。

### 限时挑战
高考实用类文本约 15 分钟完成，练到能在 10 分钟内完成全部标记 + 3 句批注，考场上就游刃有余。
