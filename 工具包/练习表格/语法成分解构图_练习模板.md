# 语法成分解构图 · 练习模板

> 适用场景：病句修改、文言文特殊句式（宾语前置/介宾后置/定语后置）、长难句理解
> 核心理念：像数学题解方程一样拆解句子结构

---

## 🧩 三步拆解法

### 第 1 步：提主干

剥离一切修饰成分（定/状/补），只保留主干。

| 成分 | 是什么 | 找法 |
|---|---|---|
| **主语（S）** | 动作的发出者 | 谁？什么？ |
| **谓语（V）** | 动作或状态 | 干了什么？是什么？ |
| **宾语（O）** | 动作的承受者 | 对谁？对什么？ |

> **口诀：** 先把"的"字定于去掉，再把"地"字状语去掉，最后把补语划掉，剩下的就是主干。

### 第 2 步：配逻辑

检验主谓宾之间的搭配是否合理。

| 病句类型 | 怎么查 |
|---|---|
| **搭配不当** | 谓语+宾语代入日常说法，看是否通顺（如："培养……水平"×） |
| **成分残缺** | 每个动词后面有没有对应的宾语？（如："具有……"后面缺什么？） |
| **句式杂糅** | 句子结构是否同时套用了两种句式？（如："是由于……造成的"） |

### 第 3 步：画结构

用树状图或括号图还原完整的修饰关系。

## 📝 练习工作单

### 示例 1：现代汉语句子

**句子：** 在全校师生的共同努力下，我校在本次科技创新大赛中取得了令人瞩目的优异成绩。

```
┌──────────────────────────────────────────────────────────┐
│ Step 1：提主干                                            │
│   我校（S）+ 取得（V）+ 成绩（O）                          │
│   → 主谓宾完整，主干通顺                                  │
│                                                           │
│ Step 2：配逻辑                                            │
│   "取得……成绩" ✓   |   "成绩"是"优异成绩" ✓              │
│   → 搭配无问题                                           │
│                                                           │
│ Step 3：画结构                                            │
│   [在……努力下]（状语）+ 我校（S）+ [在……中]（状语）       │
│   + 取得（V）+ [令人瞩目的][优异]（定语）+ 成绩（O）      │
│   → 多层状语、多层定语嵌套，但次序正确                    │
└──────────────────────────────────────────────────────────┘
```

---

### 示例 2：文言文特殊句式

**句子：** 大王来何操？（《史记·项羽本纪》）

```
┌──────────────────────────────────────────────────────────┐
│ Step 1：提主干                                            │
│   大王（S）+ 操（V）+ 何（O）                              │
│   → 现代汉语语序应是"大王操何"                            │
│   → 疑问代词"何"作宾语，前置于动词前 = **宾语前置**       │
│                                                           │
│ Step 2：配逻辑                                            │
│   "大王拿着什么" ✓                                        │
│                                                           │
│ Step 3：画结构                                            │
│   大王（S）+ 来（状语）+ 何（O 前置）+ 操（V）            │
│   规则：疑问句中，疑问代词作宾语必须前置                    │
└──────────────────────────────────────────────────────────┘
```

---

### 示例 3：病句诊断

**句子：** 通过这次学习，使我认识到自己的不足。

```
┌──────────────────────────────────────────────────────────┐
│ Step 1：提主干                                            │
│   "通过这次学习"（介词结构作状语，无主语）                 │
│   "使"（谓语）+"我"（宾语）+"认识到……"（宾补）           │
│   → 全句缺少主语！                                        │
│                                                           │
│ Step 2：诊断                                              │
│   病因：介词"通过"导致主语被淹没 = **成分残缺（缺主语）**  │
│   修改方案①：去掉"通过" → "这次学习，使我……"（主语是"学习"）│
│   修改方案②：去掉"使" → "通过这次学习，我认识到……"（主语是"我"）│
│                                                           │
│ Step 3：规律总结                                          │
│   注意"通过……使……"、"经过……让……"、"由于……导致……"        │
│   看到介词开头（通过/经过/由于/关于）立刻警惕缺主语！      │
└──────────────────────────────────────────────────────────┘
```
