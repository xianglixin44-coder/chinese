【练习1 · 信息类文本】

人工智能正在重塑教育。一方面，个性化学习系统可以根据学生的认知水平定制内容，让每个学生都能以适合自己的节奏前进；另一方面，过度依赖算法推荐可能导致"信息茧房"，使 student 只能接触自己舒适区内的知识，失去探索未知领域的机会。因此，技术赋能教育的关键不在于技术本身有多先进，而在于我们如何设计"人机协作"的教育生态。

[ ] 核心词：人工智能、个性化学习、信息茧房、人机协作

/ 层次：现象描述 / 转折（然而） / 结论

★ 主旨句：技术赋能教育的关键不在于技术本身有多先进，而在于我们如何设计"人机协作"的教育生态。

? 疑问点：作者是否暗示当前教育技术过于追求"先进"而忽略了"生态"？

【练习2 · 文学类文本】

他站在巷口，手里攥着一张对折的纸。风从巷子里灌出来，把纸角吹得一掀一掀的，像一只受伤的鸟在扑棱翅膀。他没有打开它——他已经看了太多遍。那些字像是用针尖划在纸面上的，每一个都扎眼睛。他只是在等，等巷子尽头那扇门打开，或者永远不开。

[ ] 核心意象：巷口、对折的纸、风、门——全段用物象写等待与焦虑

/ 层次：静态描写 / 动态（风掀纸角）/ 心理（他没有打开）/ 收束（等门开或不开）

★ 主旨暗示：最后句——"等巷子尽头那扇门打开，或者永远不开"完成了一种悬而未决的焦虑

? 反常处：为什么"每一个都扎眼睛"？以痛觉写视觉，强化情感冲击

▲ 品味句："像一只受伤的鸟在扑棱翅膀"，比喻纸角掀动——赋物以生命

📝 文言文翻译练习

翻译下列句子，注意加粗字词和特殊句式。

1. 群臣吏民能面刺寡人之过者，受上赏。

翻译：群臣、官吏和百姓，能够当面指责我的过失的，给予上等奖赏。

考点：「面」— 名词作状语，当面（而非脸面）。

2. 大王来何操？

翻译：大王来时带了什么？

考点：疑问代词「何」作宾语前置 = 宾语前置句（正常语序：大王来操何？）

3. 故余虽愚，卒获有所闻。

翻译：所以我即使愚笨，最终还是有所收获。

考点：「虽」— 即使（表让步，≠虽然）；「卒」— 最终（≠死亡/兵卒）。

4. 下列句子中，属于介宾后置的一句是：

✅ 正确答案：B（战于长勺 = 在长勺作战）

A = 宾语前置 | C = 定语后置 | D = 被动句

📝 病句专项练习

判断下列句子是否有语病，若有，指出问题类型。

1. 通过这次社会实践，使同学们对基层工作有了更深刻的理解。

✅ 正确答案：B（成分残缺·缺主语）

介词"通过"淹没了主语。修改：去掉"通过"→"这次社会实践，使同学们……"；或去掉"使"→"通过这次社会实践，同学们对基层工作有了更深刻的理解。"

2. 我们必须不断提高和培养自己的创新能力。

✅ 正确答案：A（没有语病）

⚠️ 易误判为搭配不当。检验：「提高……能力」✓；「培养……能力」✓ 两者均成立。若改为"提高和培养水平"，则"培养水平"不成立。

3. 这种做法，广为人民群众所欢迎。

✅ 正确答案：C（句式杂糅）

白话"广为" + 文言被动"为……所……" 混用。
修改："这种做法，受到人民群众的广泛欢迎。"

4. 他的写作水平有了明显的提高和改善。

✅ 正确答案：B（搭配不当）

「提高……水平」✓；「改善……水平」✗（应为"改善状况"）。"提高和改善"不能共用同一宾语"水平"。

5. 为了防止这类事故不再发生，学校加强了安全教育。

✅ 正确答案：B（否定不当）

"防止"已含否定义，"不再"又否定，双重否定变肯定，逻辑反了。
修改：去掉"不"→"为了防止这类事故再次发生"；或改为"为了确保这类事故不再发生"。

📝 作文真题 · 模板实战

以下为近年高考思辨类作文真题。用「他们说/我说」模板构思一段核心议论段落。

【真题1】 有人说，人工智能将取代人类的大部分工作；也有人说，人工智能只是工具，关键在人如何使用。对此你怎么看？请写一段话。

A1 诚然，AI取代重复性劳动的趋势在某种程度上具有其合理性，例如工厂流水线和基础数据录入；然而，如果我们审视人类工作的本质——创造、共情与价值判断——就会发现，AI目前触及的只是"执行层"，而非"决策层"。

B1 这并非偶然的巧合，而是技术发展的必然阶段。究其本质，是因为机器善于"优化已知"，而人类的价值在于"探索未知"。

C1 由此，我们不仅看到了技术赋能的高效，更应当警惕将"人"降格为"被替代品"的思维惯性。

【真题2】 "知之者不如好之者，好之者不如乐之者。"兴趣是最好的老师。但也有人说，成功更多需要毅力和纪律。请写一段议论文段落。

A2 有人或许会主张：成功更多依赖毅力和纪律，兴趣不过是起点。这种看法并非全无道理，但它忽略了一个关键前提——没有内在兴趣驱动的"毅力"，往往沦为机械的自我强迫，难以持久。

B2 事实上，兴趣与毅力并非对立。不仅体现在"乐之者"天然的投入度更高，更深刻地反映在——兴趣本身就能持续生成"坚持下去的理由"。

C2 或许，最好的答案并不在于二选一，而在于我们在教育设计上寻得一种平衡——用兴趣点燃起点，用纪律守护终点。

【真题3】 材料：移动互联网时代，"碎片化"已成为常态——碎片化阅读、碎片化社交、碎片化时间。有人担忧这种趋势正在消解深度思考能力；也有人认为碎片化是效率的体现。对此你有怎样的思考？

A3 站在效率论者的立场来看，碎片化确实是信息时代的合理适应。但当我们把视野放宽到人类文明的纵深——科学突破、哲学体系、文学巨著——就会发现，几乎所有深度创造都诞生于"整块时间"的沉浸之中。

B4 从更根本的层面来看，碎片化之所以值得警惕，是因为它触及了一个被普遍忽略的前提——人的认知带宽是有限的。每一次切换都在消耗注意力残留。

C4 归根结底，碎片化与深度思考的争论所关切的，不仅仅是阅读习惯的问题，更关乎每一个人在信息爆炸的时代里能否守住思想的完整性。

🗣️ 格莱斯会话含义解码法（言外之意）

语用学 · 格莱斯"会话含义理论" → 解答"分析画线句深层含意"

公式：言外之意 = 字面意义 + 被违反的语用准则 + 说话者真实意图
准则① 量的准则 信息量过多或过少 = 刻意回避
准则② 质的准则 故意说反话 = 讽刺或无奈
准则③ 关联准则 答非所问 = 转移话题或暗示

📊 多文本对话矩阵法（比较阅读）

多元读写理论 → 多文本对比维度：核心观点 / 论据组织 / 情感基调

维度文本A（庄子）文本B（荀子）冲突/对话点
核心观点顺应自然，无为而治制天命而用之，人定胜天人与自然关系的根本对立
论据组织寓言故事、神话想象（虚）礼乐制度、社会事实（实）逻辑论证手段的虚实差异
情感基调超脱、逍遥、静谧积极、刚健、理性知识分子对时代困境的不同救赎

⚔️ 图尔敏反驳操练法（论证拆解）

图尔敏论证模型 → 攻克"指出逻辑漏洞并反驳"题型

① 解构对方论证
主张(Claim) + 证据(Data) + 前提(Warrant)
② 寻找逻辑漏洞
证据孤立性？前提何时不成立？
③ 构建反驳链条
反证数据(Counter-Data) → 推翻前提 → 对方主张崩溃
示例：对方主张"AI将取代作家"→ 前提"文学=词汇排列"→ 反驳"文学核心是情感共鸣，AI无法产生人类体验"→ 前提被推翻

🎨 文体风格基因映射法（风格比较）

比较两个文学作品风格异同 → 三维修辞基因图谱

词汇密度
句式长短、修饰成分
修辞色彩
素朴 ↔ 绚丽
情感态度
冷峻批判 ↔ 温情牧歌

例：鲁迅《故乡》的冷峻白描 vs 沈从文《边城》的田园牧歌，同写乡村，基因迥异。

📝 写作评估量规（60分·水平四标准）

层级论证思维语言表达
50-60分辩证反驳：主动引入反面观点(Concession)，精准找出前提局限(Rebuttal)，完成高维逻辑整合反问、双关等高级修辞，语境契合度高
