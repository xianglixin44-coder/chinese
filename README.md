# 语文提高训练 · 西方方法映射图谱

> 将经过验证的西方思维方法论，转化为中学语文可操作的日常训练体系。

## 📂 项目结构

```
语文提高训练/
├── README.md                        ← 本文件（总纲与使用说明）
├── 教材推荐书单.md                   ← 教材/参考书详解与使用场景
├── 训练计划与执行方案.md              ← 每日/每周训练节奏
│
├── 工具包/
│   ├── 模板卡片/
│   │   ├── 他们说我说_逻辑连词模板卡.md   ← 议论文段落过渡模板
│   │   └── 主动标记阅读法_符号规范表.md   ← 阅读标注符号系统
│   │
│   ├── 练习表格/
│   │   └── 语法成分解构图_练习模板.md     ← 长难句拆解工作单
│   │
│   └── 闪卡数据集/
│       ├── 文言文核心实词_50词.csv        ← Anki 可导入闪卡（实词）
│       ├── 文言文核心虚词_18词.csv        ← Anki 可导入闪卡（虚词）
│       └── 文学常识_20条.csv             ← Anki 可导入闪卡
│
└── outputs/                         ← 生成的练习材料（gitignored）
```

## 🧠 核心理念

把语文从"感悟式学习"转变为 **可拆解、可训练、可反馈** 的技能习得。

| 西方方法论源头 | 映射到语文场景 | 对应工具 |
|---|---|---|
| 分析阅读（艾德勒） | 现代文阅读结构化拆解 | 主动标记阅读法 |
| 论证模型（格拉夫） | 议论文起承转合 | "他们说/我说"模板卡 |
| 结构主义语法（索绪尔→朱德熙） | 病句/文言句式分析 | 语法成分解构图 |
| 间隔重复（Spaced Repetition） | 文言实词/文学常识记忆 | 费曼闪卡（Anki） |

## 🚀 快速开始

1. **通读**「教材推荐书单.md」— 选择 1–2 本核心读物
2. **打印**「主动标记阅读法_符号规范表.md」— 下次阅读就贴在手边
3. **导入** 闪卡 CSV 到 Anki — 每天 5 分钟刷卡
4. **用**「语法成分解构图_练习模板.md」— 每周拆 3–5 个长难句
5. **套用**「他们说/我说逻辑连词模板」— 每次作文练习强制使用
